== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `awwm-slider.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[awwm-slider]` shortcode in any page or post where you need to display slider.
4. Install contact form 7 wordpress plugin.
5. Import contact-form.xml file is available inside plugin folder.
6. create custom posts under AWWM SLider or import demo-posts.xml.


== Modifications ==

   In case of future development or modification 

1. For Frontend changes go to /awwm-slider/public folder.
2. For Admin changes go to  /awwm-slider/admin folder.
